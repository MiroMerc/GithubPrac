This project is meant to display air traffic data for airports from the years 2020-2025
It is up to date on passenger data from the year 2025, as of 4/22/2026
All data is sourced from (https://en.wikipedia.org/wiki/List_of_busiest_airports_by_passenger_traffic)
The current plan for the project is to add data as the years come, and to improve readability and UI
The repo is organized with the QMD file and it's current output file in html form, along with it converted into pdf from html
